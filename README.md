# HoverCraft3000
This repository contains the documentation for the HoverCraft3000 project.

## Installation
Install [Python](https://www.python.org/downloads/) onto your computer.
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install python libraries.

Ensure you have the following libraries downloaded by running the following commands in terminal:
```bash
pip install tk
```
